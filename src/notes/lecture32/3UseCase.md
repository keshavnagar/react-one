# use casees 
1. feature toggle: use the toggle switch to enable or disable features in our app
2. dark mode: implement a dark mode toggle switch to switch between light and dark mode
3. setting: use toggle switches in a setting menu to turn various setting on or off
4. visibility control: toggle the visibility of certain ui elements or sections
5. form inputs: use toggle switches as part of form inputs to capture user preferences.

# features
1. a switch that toggles between on and off states.
2. change the appearance based on the current state(eg. different colors for on and off)
3. display the current state (eg. "on" or "off")
